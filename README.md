# Challenge Rain chart implementation.

## How to install

1. Install the project
 ```
 npm install
 ```
2. Run locally, and open your browser at http://localhost:8080/
 ```
 npm run dev
 ```
3. Run tests
 ```
 npm run test
 ```
4. Generate Production bundle
 ```
 npm run build
 ```
